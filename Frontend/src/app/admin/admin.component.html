Nella GESTIONE ACCOUNT è possibile creare un nuovo profilo utente, specificando Username (per convenzione, un indirizzo mail in un formato valido), Password (lunghezza non inferiore agli 8 caratteri) e tipologia (Utente standard o Amministratore). E' inoltre possibile modificare i dati di un profilo utente esistente o eliminare un profilo utente non più utilizzato.
Gestione Account
Nella GESTIONE IMPIANTI è possibile creare un nuovo impianto, specificando il nome dell'impianto ed il cliente proprietario. E' inoltre possibile modificare i dati di un impianto esistente o eliminare un impianto non più attivo.
Gestione Impianti
Nella GESTIONE SENSORI è possibile creare nuovi sensori, specificando grandezza fisica rilevata (tra le 6 disponibili) e marca del produttore. E' inoltre possibile eliminare sensori esistenti.
Gestione Sensori
